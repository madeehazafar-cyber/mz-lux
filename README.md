# MZ Lux

## Purpose

MZ Lux is a styling web app designed to help users quickly choose what to wear. Many people struggle with matching clothing for different occasions or creating outfits that reflect their personal vibe. MZ Lux solves that problem by allowing users to mix and match digital clothing items and receive outfit recommendations.

## Target Audience

This app is mainly designed for teenagers and young adults who are interested in fashion, styling, and social media aesthetics. It also serves students who want quick outfit inspiration without spending too much time deciding what to wear.

## Key Features

- **Outfit Generator:** Users can select clothing categories such as tops, pants, skirts, shoes, and accessories to create complete outfits. The app can randomly generate outfit combinations.
- **Fashion Theme Filters:** Users can filter outfits by categories like Streetwear, Old Money, Minimalist, Casual, Formal, and more.
- **Save Option:** Users can save outfit options they like into a personal collection to revisit later.
- **Interactive Visual Layout:** The app displays clothing pieces with images and sections that are visually appealing and easy to use.

## Design Plan

MZ Lux will have a clean, easy-to-navigate layout with a luxurious look.

### Layout

- Homepage with the MZ Lux logo and navigation bar.
- Main outfit builder at the center of the screen.
- Clothing categories like tops, bottoms, shoes, and accessories in separate sections.
- Buttons for generating outfits and saving favorite combinations.
- A favourites page to view saved outfits.

### Style and Color Palette

- Neutral colors such as black, white, and beige.
- Rounded buttons and soft gradients.
- Elegant spacing and visuals for a luxurious aesthetic.
- Montserrat font for a modern, professional feel.

## Project Structure

- `index.html` — Homepage and main landing experience.
- `categories.html` — Outfit builder and fashion categories.
- `themes.html` — Theme-based outfit exploration.
- `favourites.html` — Saved outfits and favourites.
- `style.css` — Global styles, branded layout, and premium visuals.
- `main.js` — Interactive behavior for outfit generation and theme selection.

## Why MZ Lux?

MZ Lux aims to make styling faster and more fun by combining outfit recommendations with a stylish, fashion-forward user experience. It helps users feel confident in their outfit choices and discover new looks that match their personal style.
